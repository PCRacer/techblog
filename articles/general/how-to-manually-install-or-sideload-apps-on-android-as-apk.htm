How to manually install or sideload apps on Android as APKs

Google's Playstore offers millions of apps, but due to censorship, Google's anti-competitiveness, and platform restrictions, not every good app can be found in the store. So the only way to get such apps is by sideloading or manually installing them on your Android device.

Thanks to the openness of the Android system, you can sideload and install Android apps from third-party stores or directly from our website. We are very happy to announce that we have also now entered the Android ecosystem with three great and very useful Android apps.

It’s surprisingly easy to install Android apps from outside the Google PlayStore, but there are some things you should know before getting started...


Why Sideloading Is Disabled by Default

Before we get started, let’s talk about why sideloading is disabled by default. The Google PlayStore has security protocols to make sure the apps you download are safe. While not perfect, it’s a pretty decent system and keeps everybody safe from harmful apps. Sideloading on the other hand let's software developers provide you with free apps without the hassle of Google oversight.

It’s very important to make sure you get apps from trusted sources or vendors if you’re going to go outside the PlayStore. Since we've been in the software business for over twenty years now, there's no need for you NOT to trust us. We only have the best free software apps available for you to enjoy, on any Windows PC, web browser, or Android device.

Before you can sideload an app on Android, you need to obtain its installation file. Android app installers have the .APK file extension that's short for Android Package. APK files contain everything that's needed to install and run the app.


Transferring the APK file to your phone

If you haven’t downloaded the APK file directly onto your Android phone, you’ll need to transfer it from your computer to your phone. You can transfer it using Bluetooth, WiFi, a USB cable, or a cloud storage service.



Manually Installing or Sideloading your APK file

  1. Once the APK has been transferred/downloaded to your phone, you're ready to install it.
  2. Locate the file on your phone. The downloaded APKs can be found in the Downloads folder. Or you can use a file manager to navigate to the Downloads folder.
  3. Tap or hold your finger on the file name to start the installation. If this is the first time you’re installing an APK from the file manager, the Downloads folder, or the web browser, Android will throw a warning and ask your permission to continue. Give the permission and, depending on the app from where you’re installing the APK, you may have to tap back to get to the installer. In some apps, you’ll automatically be sent back to the installer.
  4. Tap on Install, and the application will be installed successfully unless there are compatibility issues.

If your Android device is running on Nougat or an older version, you’ll first have to enable Installation from Unknown Sources by going to Settings >> Security, after which you'll be able to install the APK. Sideloading or manually installation support is available on all Android versions, and you don’t need root access or any other special privileges.

So there you go, simply download your favourite app onto your device in APK form, tap and hold on the file, and Android will install the file for you. Couldn't be any simpler!
